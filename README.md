  ## Description
  Our motivation behind building the Tech Haven E-Commerce Store was to create a cutting-edge platform that offers a seamless and enjoyable shopping experience for tech enthusiasts. We wanted to provide a curated selection of high-quality tech products while leveraging modern technologies to ensure fast performance, security, and user satisfaction.

  We built the Tech Haven E-Commerce Store to meet the growing demand for a reliable and user-friendly online platform for purchasing tech products. Our goal was to create a centralized hub where customers can easily find and purchase the latest gadgets, electronics, and accessories while enjoying a smooth shopping experience.

  Tech Haven E-Commerce Store solves the problem of fragmented online shopping experiences for tech products. By offering a wide range of gadgets, electronics, and accessories in one convenient platform, it eliminates the need for customers to visit multiple websites to fulfill their tech needs. This centralized solution simplifies the shopping process, saving users time and effort while providing access to the latest tech innovations.
 

  ## Installation
  1. Ensure you have MongoDB, Node.js, and Docker installed on your system.
  2. Clone the project repository from GitHub.
  3. Navigate to the project directory in your terminal.
  4. Run `npm install` to install project dependencies.
  5. Set up environment variables for MongoDB connection, Stripe API keys, and any other required         configurations.
  6. Run `npm run dev` up to spin up the Docker containers and start the server.
  Navigate to the specified URL in your browser to access the Tech Haven E-Commerce Store.

  ## Usage
  1. Setting up an online store: You can customize and configure the store to sell a wide range of tech products, including gadgets, accessories, and more.

  2. Learning: Developers can explore the codebase to understand how to build an e-commerce platform using modern technologies like Next.js, React, TypeScript, MongoDB, and GraphQL.

  3. Experimentation: You can experiment with different features, integrations, and designs to enhance your understanding of e-commerce development and improve your skills.

  4. Deployment: Once customized to your requirements, you can deploy the project to a hosting platform to launch your own e-commerce store and start selling products online.
